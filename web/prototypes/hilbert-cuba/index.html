How to read this

Compact view (left): Cuba's IPv4 prefixes packed into a dense Hilbert fingerprint. Each cell ≈ 4 IPs. This is a remapped view for readability — IP blocks that are numerically close appear near each other. Brighter = visible from selected vantage.

Global view (right): Full IPv4 Hilbert at /20 granularity. Shows where Cuba's allocations actually sit among all 4 billion IPv4 addresses. Useful for seeing how scattered or concentrated a country's space is globally.

Important: These are IP address space maps, not physical maps. AS paths shown are logical BGP paths, not fiber routes.
